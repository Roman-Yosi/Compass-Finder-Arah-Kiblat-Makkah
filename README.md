# Compass Finder Arah Kiblat Makkah

Website pencari arah kiblat berbasis GPS dan sensor kompas perangkat.

## Fitur

- Menampilkan arah kiblat ke Makkah
- Menggunakan GPS
- Menggunakan sensor kompas perangkat
- Responsif untuk Android dan iPhone
- Siap digunakan di GitHub Pages

## Deploy

1. Upload semua file ke repository GitHub.
2. Buka Settings → Pages.
3. Source: Deploy from Branch.
4. Branch: main.
5. Save.
